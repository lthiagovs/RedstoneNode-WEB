Sobre o Controlador de Bots do Minecraft

Este projeto tem como objetivo criar uma interface de controle para bots no jogo Minecraft. Com ele, você pode facilmente gerenciar e interagir com seus bots de maneira simples e eficiente.

O controlador oferece várias funcionalidades para facilitar a administração dos bots, garantindo uma experiência de jogo mais personalizada.
